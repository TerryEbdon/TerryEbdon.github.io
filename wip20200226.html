Dark Mode

I run everything in Dark Mode, where possible. But some apps don't support it, or misbehave with it. e.g. Microsoft Outlook can mess up the calendar view when Dark Mode is enabled. Today Windows prompted me to enable the OS Dark Mode configuration. I did, and it works well. This prompted me to check a few other applications. Atom was already working great with the "One Dark" syntax and UI themes. I switched SourceTree and GitHub Desktop over, without problem. But what about FreeCommander? This is my most heavily used app, and it has no built-in dark theme. I've tried creating one before, it didn't work out. Today I succeeded, with FreeCommander XE 2016.

Free Commander XE colours

To get this working I needed a font colour that works with three background colour. This needs to be configured in several places. It's easy to do, as long as you remember every place that needs configuring. I've still got a couple of things to change, but the following settings will be good enough for the time being.

GitHub Desktop can't clone

GitHub's Desktop app is refusing to clone repositories. It claims the revocation server is offline. From a quick web search it seems likely that my ISP, or someone else, is blocking the request.
