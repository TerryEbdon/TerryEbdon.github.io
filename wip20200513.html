Atom spell checking

Within the last week or so spell checking stopped working in Atom. This is very annoying, as I use Atom for editing large MarkDown sites. My Windows locale is en-GB and Atom is set up to use the system locale. Windows Settings shows the locale is properly configured. Atom is up to date and there's no pending Windows language pack updates in the Windows Store. There was a restart pending, to install the latest Windows update, but an update and restart made no difference.

Changing editors

I previously used VS Code. I found it very slightly better than Atom, but was happy using either. I switched to using Atom exclusively, a year or two back, when I had to fall back to using my old, 32-bit, Windows 7 laptop for a few months. I found VS Code to be unreliable on that machine, though it worked fine on my 64-bit Windows 7 tablet. Now that I'm back at home, using a Windows 10 desktop almost exclusively, there's no need to avoid VS Code. Rather odd that Code doesn't have a built-in spelling checker, but the Code Spell Check extension seems pretty good. Better than the Atom spell checker. Problem solved. Now I just need VS Code extensions equivalents to my Atom plugins.
