Doxygen Debugging

DevDiary now works as a WebDoxy project again! I moved the output folder to be below the WebDoxy/Projects/DevDiary folder, it was previously under the WebDoxy/output folder. Again this demonstrates that the issue is likely in Doxygen. It probably wouldn't be that hard to track down in the Doxygen C++ source code, but my time is better spent progressing my own projects.

Using Mercurial from Atom

I had the atom-hg package installed but it did nothing. There are suggestions in atom-hg issues about putting Hg on the path and restarting Windows. But atom-hg has been installed for months and Hg has been on the path for years. Reinstalling atom-hg made no difference. I uninstalled it and used tortoise-hg instead. This worked immediately.
